Manage git branches for **$ARGUMENTS**:

List branches:
- Run git branch -a (all branches)
- Run git branch -vv (with tracking info)

Create new branch:
- Run git checkout -b <branch-name>
- Run git push -u origin <branch-name>

Switch branches:
- Run git checkout <branch-name>
- Run git status to verify

Report current branch and available branches.
