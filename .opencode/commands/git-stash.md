Manage git stash:

Stash current changes:
- Run git stash push -m "message"
- Run git status to verify clean working tree

List stashes:
- Run git stash list

Apply stash:
- Run git stash pop (apply and remove)
- Run git stash apply (apply and keep)

Report stash status and contents.
